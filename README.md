# srdump

Segment Routing Debugger

## Usage

```
$ git clone <this-repo> && cd _
$ ./bootstrap.sh
$ make && sudo make install
```

for TiNet users
```
postinit:
  - cmds:
    - cmd: make -C /home/vagrant/git/srdump install_docker
```
